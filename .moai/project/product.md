# TechPulse Blog — Product Document

> 生成日: 2026-03-16
> バージョン: 0.1.0

---

## 1. プロジェクト概要

TechPulse Blogは**個人技術ブログシステム**です。
ObsidianのマークダウンファイルをRSSで収集したIT技術トレンドと統合し、
Go + HTMX + Tailwind CSSで構築された軽量なブログとしてレンダリングします。

- **バックエンド**: Go (Echo) + SQLite + Bleve全文検索
- **フロントエンド**: templ + HTMX + Tailwind CSS
- **コンテンツ**: Obsidianマークダウン + RSS収集パイプライン
- **インフラ**: Oracle Cloud Always Free Tier

## 3. 核心機能

### Phase 1: ブログエンジン（MVP）

| 機能 | 説明 |
|------|------|
| マークダウンパース | goldmark + frontmatter処理 |
| ブログレンダリング | templ + HTMX（記事一覧、詳細、ページネーション） |
| スタイリング | Tailwind CSS |
| コードハイライト | サーバーサイドハイライティング |
| Mermaid対応 | クライアントレンダリング |
| RSSフィード | フィード生成 |

### Phase 2: Obsidian連携

| 機能 | 説明 |
|------|------|
| ファイル監視 | Obsidian Vaultのマークダウン変更検知 |
| 自動変換 | frontmatter自動付与、スラッグ生成 |

### Phase 3: RSS収集パイプライン

| 機能 | 説明 |
|------|------|
| RSS取得 | 複数フィードの定期取得 |
| 要約生成 | 記事の自動要約 |

### Phase 4: 技術エクスプローラー

| 機能 | 説明 |
|------|------|
| トレンド分析 | 技術トレンドの可視化 |
| 検索 | Bleve全文検索 |

### Phase 5-6: デプロイ・ドメイン

| 機能 | 説明 |
|------|------|
| デプロイ | Oracle Cloud Always Free |
| ドメイン | カスタムドメイン・DNS設定 |
